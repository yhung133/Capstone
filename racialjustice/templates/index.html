Racial and Criminal Justice in Champaign County


Overview

Racial Disparities exist in the criminal justice system at nearly every stage from arrest rates, bail amounts, to incarceration lengths and sentence results. This website and its behind database are built to:
1) Collect and centralize scattered data in Champaign criminal justice system
2) Visualize and publish collected data to help policymaking


Visite Previous Web App


Background

In Fall 2016 semester, fifteen UIUC students worked with the Champaign RJTF to collect criminal justice data and build web-based visualization tools.
Students are divided into three working groups – police, jail and court. Each group achieved building a dashboard using R.
Finally, Prof. Wilson incorporated three groups’ work into one web application.


Watch Presentation on Youtube


Current Progress

In Spring 2017, one student from the police group – Yuyan Huang continued working on this project as her master of urban planning capstone project. She built this website, which will be maintained and refined by Champaign Racial Justice Task Force, NCSA, and upcoming interested students.
